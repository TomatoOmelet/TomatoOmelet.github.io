    Lobster Revolution                                                              2018 Fall
    Programmer on a 7-member team

    This is a card game about lobsters! We have been working on this 
    project for a whole month and I am working on the battle system. 
    I have never maken digital card games before, so I enjoy programming 
    this game.

    The first thing I tried to implement is dragging the card to floor. 
    I use raycast to check what player touches when they press down and 
    release mouse, so I can know if they are clicking a card and if they 
    are dragging it to a spot on the floor. Later there are more things 
    to test, like commanding lobsters to attack/denfense and choosing 
    targets/materials. I add variables to store the current states of 
    the game and test different stuffs when player press down/release 
    mouse button at different states.

    Chibi Inu                                                                   2018 Fall - Present
    Programmer on a 5-member team

    It is a 2D platformer game about a dog. I joined this project in 
    the middle, so I spent many time reading and understanding existing 
    code when I wanted to program something related to an existed system.
    
    Even though it is a 2D platformer game, the mechanic is already 
    implemented when I joined the team. I work on things like save system, 
    dialogue system and cutscene. I also made the UI to create and select
    save file at Main Menu. For the dialogue system, I stored the dialogue 
    information in json files to make it easier for localization if we want
    to do that in the future. My teammates are skillful at their fields, so I 
    have a good feeling about this project.
